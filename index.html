Estilização de Títulos com especificidade:

Quando selecionamos no css o h1 mais de uma vez, o último código prevalecerá, isso chamamos de efeito cascata. Porém, quando usamos uma classe com "id", independente do último código digitado, o id prevalecerá, pois tem maior especificidade.

Diferença de especificidade entre CLASSE e ID:

Id:

  • Identificador único
  • Só pode ser usado uma vez por página.
  • Ideal para elementos específicos (ex: um botão único)

Classe

  • Pode ser usada em vários elementos.
  • Serve para aplicar estilos ou comportamentos repetidos.
  • Mais flexível e reutilizável

Qual é mais indicado para usar?

Usar classe no HTML é geralmente melhor porque oferece mais flexibilidade e reutilização no código. Enquanto o id serve para identificar um único elemento, a classe pode ser aplicada a vários, permitindo que estilos e comportamentos sejam compartilhados entre eles.

Por que é melhor usar classe?

  • Facilita a manutenção do código.
  • Permite aplicar o mesmo estilo a vários elementos.
  • Evita conflitos e limitações que o id pode causar.

Combinadores:

Combinadores são elementos do CSS que definem relações entre os seletores, ou seja, como um elemento está conectado a outro dentro da estrutura do HTML.

Exemplo:

  • Espaço ( ) -seleciona descendentes (ex: div p)
  • Maior (>) - seleciona filhos diretos (ex: ul > li)
  • Mais (+) - seleciona o irmão imediato (ex: h1 + p)
  • Til (~) - seleciona irmãos posteriores (ex: h1 ~ p)

Pseudoclasse:

Uma pseudoclasse é usada para aplicar estilos a elementos com base em seu estado ou posição, sem precisar adicionar classes no HTML.

Exemplo:

  • p:first-child seleciona o primeiro filho
  • p:nth-child(3) aplica estilo ao terceiro filho

Box Model / Display

Border:

A border é a borda que envolve o conteúdo e o padding de um elemento no Box Model do CSS.

  • Cria uma moldura visual ao redor do elemento.
  • Pode ter espessura, cor e estilo (solid, dotted, dashed...).
  • A border aumenta o tamanho total do elemento (salvo com box-sizing: border-box;)

Padding:

  • Espaçamento interno entre conteúdo e borda.
  • Aumenta a área de fundo sem alterar o conteúdo.
  • Pode ser definido em px, em, rem, %...

Shorthand:

  • padding: 10px; → todos os lados
  • padding: 10px 20px; → cima/baixo e lados
  • padding: 10px 20px 30px 40px; → topo, direita, baixo, esquerda
